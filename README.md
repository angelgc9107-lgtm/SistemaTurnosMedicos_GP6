# SistemaTurnosMedicos - Diseño Orientado a Objetos

- **Materia:** Diseño Orientado a Objetos
- **Carrera:** Tec. en Programación de Sistemas
- **Grupo:** Grupo N° 2

## Introducción

Este proyecto tiene como objetivo desarrollar el diseño orientado 
a objetos de un sistema para un consultorio médico que busca 
digitalizar la gestión de sus turnos.
